# Customer Churn Prediction

This project predicts whether a customer is likely to leave a service using machine learning techniques.

##  Overview
The model is built using customer data and applies data preprocessing, feature engineering, and classification algorithms to predict churn.

##  Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

##  Features
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Feature engineering
- Model training and evaluation

##  Files
- `customer_churn.ipynb` → Main notebook with complete workflow  
- `customer_churn.csv` → Dataset used for training

1. Open the notebook:
   ```bash
   jupyter notebook customer_churn.ipynb
